FAQs (and hypothetical 'What ifs')

All that you ever wanted to know about me.
General Quiz Privacy Bugs & Problems

1. What am I doing here?

You somehow stumbled on this amazing webapp (or probably relentlessly pestered into checking it out by a certain someone) and got curious and clicked on the huge link on top.

2. Oh, okay. Wassup?

I'm super-excited about this. Basically, this is a web application that lets you take a time-based quiz everyday.

3. Honestly, that doesn't seem exciting at all.

But wait! There's more! The quiz also comes with a company-wide ranking system. You can view a bunch of statistics (yes, before you ask, we have graphs) about the day's quiz in a very cool page in real-time too! The app is built with a simplistic design aesthetic while also giving you the option to pick a custom skin that the app remembers the next time you log in!

4. Can I also take the quiz on my iPad/iPhone/<insert mobile device name here>?

Absolutely. The website is fully responsive and scales beautifully across a wide range of resolutions.

5. OH MY GOD AMAZING TELL ME MORE!

I'm gonna geek out a little, but stay with me here. This entire app was built using JavaScript technology! The backend of the app was coded in Node.js and the interface using plain old HTML, CSS and JavaScript (of course, with the help of Bootstrap & jQuery. Even the database system, MongoDB, uses JSON for storing all the data.

6. That's awesome.

I know, right?

1. How do I take the quiz?

A daily quiz (of usually 5 to 10 questions) is hosted every day, between {{ start_hour|json|replace(',', ':', 'g') }} and {{ stop_hour|json|replace(',', ':', 'g') }} hours. All you have to do is login and start the quiz. Each question has an intelligently-calculated time limit, and the time left is displayed by an animated progress bar. If you do not select an answer within the time period, an empty response is recorded and the page automatically switches to the next question.

2. What's this quiz like?

Each question has 3 to 5 multiple choices. Some might have images too. When you select one of the choices, you are shown the next question. At the end of the quiz, you can review your answers.

3. How does the ranking system work?

It's simple - each question carries 1 point. Rankings are based on total points and the average response time (rounded off to 3 decimal points).

1. Wait, you actually have a privacy policy?

Yup. The only personal information we ask of you is your password. This password is one-way encrypted using a super secure algorithm - meaning a hash value is saved along with a unique salt. This means that even if someone managed to gain access to the database, he or she cannot view your password. Read more about how one-way hashes work here.

1. I found a bug/My house burned down/My girlfriend left me.

That is heart-breaking. Please report any bugs you found to your friendly neighbourhood developer via email or ping.
